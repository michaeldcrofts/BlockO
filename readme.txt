Thanks for looking at BlockO.
It is designed to execute the Python code directly on the host machine, not simulate it, so that physical inputs and outputs of a Raspberry Pi can be used. 
Therefore you need to have python installed and you need to start the Python server.
##################################################################
################## Set-up instructions ###########################
#  1) Install Python, if not already installed                   #
#  2) Open server.py and set it running                          #
#  3) Open a web browser and navigate to http://127.0.0.1:8081   #
#  4) Start coding in either Blockly or Python                   #
